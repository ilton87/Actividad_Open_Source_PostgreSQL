# Introducción



PostgreSQL es uno de los sistemas de gestión de bases de datos relacionales más utilizados en el mundo debido a su estabilidad, seguridad y capacidad para manejar grandes volúmenes de información. Al ser un software de código abierto, es ampliamente utilizado tanto en entornos académicos como empresariales.



Las herramientas de administración de PostgreSQL permiten facilitar tareas como la creación de bases de datos, ejecución de consultas SQL, administración de usuarios, monitoreo del rendimiento y realización de copias de seguridad. Estas herramientas mejoran la productividad de administradores y desarrolladores al proporcionar interfaces gráficas o web para interactuar con el sistema.



Por otra parte, el software Open Source se caracteriza por permitir el acceso al código fuente, posibilitando su estudio, modificación y distribución por parte de la comunidad. Gracias a ello existen numerosas herramientas gratuitas para la administración de PostgreSQL.



# Metodología



Para la realización de esta investigación se llevó a cabo una búsqueda documental utilizando fuentes confiables relacionadas con la administración de bases de datos PostgreSQL. Como criterios de selección se consideraron la licencia de uso, el tipo de interfaz (web o gráfica de escritorio), la disponibilidad de documentación oficial, la facilidad de instalación y las funcionalidades ofrecidas para la administración y monitoreo de bases de datos. Finalmente, se realizó una comparación de las herramientas seleccionadas para identificar sus principales características y ventajas.



# Desarrollo



## 1. phpPgAdmin



**Licencia:** GPL



**Tipo de interfaz:** Web



**Sitio oficial:** https://github.com/phppgadmin/phppgadmin



**Descripción:**



phpPgAdmin es una herramienta web desarrollada específicamente para la administración de bases de datos PostgreSQL. Permite crear bases de datos, gestionar tablas, usuarios, vistas y secuencias mediante una interfaz accesible desde cualquier navegador.



## 2. DBeaver Community Edition



**Licencia:** Apache License 2.0



**Tipo:** GUI



**Sitio Oficial:** https://dbeaver.io



**Descripción:**



DBeaver Community Edition es una herramienta de código abierto que permite conectarse a PostgreSQL y a otros sistemas de gestión de bases de datos. Cuenta con un editor SQL avanzado y herramientas de importación y exportación de datos.



## 3. Adminer



**Licencia:** Apache License 2.0



**Tipo:** Web



**Sitio Oficial:** https://www.adminer.org



**Descripción:**



Adminer es una aplicación web ligera para la administración de bases de datos mediante navegador. Permite gestionar tablas, usuarios, índices y ejecutar consultas SQL de manera rápida y eficiente.



## 4. Beekeeper Studio



**Licencia:** GPL v3



**Tipo:** GUI



**Sitio Oficial:** https://www.beekeeperstudio.io



**Descripción:**



Beekeeper Studio es un cliente SQL moderno y de código abierto diseñado para facilitar la interacción con diferentes sistemas de bases de datos mediante una interfaz amigable e intuitiva.



## 5. OmniDB



**Licencia:** MIT



**Tipo:** Web



**Sitio Oficial:** https://omnidb.org



**Descripción:**



OmniDB es una herramienta web enfocada en el desarrollo y administración de PostgreSQL. Incorpora un editor SQL inteligente y herramientas de monitoreo para mejorar la productividad.
